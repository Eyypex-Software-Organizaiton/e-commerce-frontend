## İndirme İşlemleri


### Backend

```bash 
git clone project_name
```

```bash 
cd e-commerce-frontend


<!-- We use yarn but firstly we need to install npm -->
<!-- If you already have npm, you don't need to install it. -->
npm install -g npm

npm install --global yarn

yarn

yarn start
```

http://localhost:3000/
